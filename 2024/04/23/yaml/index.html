Yaml

YAML：YAML Ain’t Markup Language，也就是说YAML是一种非标记语言。

1.基本语法

  • 大写敏感
  • 使用缩进表示层级关系
  • 缩进不允许使用tab,允许空格(这个要特别注意)
  • 缩进的空格数不重要，只要相同层级的元素左对齐即可

2.数据类型

YAML支持以下几种数据类型:

  • **对象: ** 键值对的集合，又称为映射(mapping) /哈希(hashes) / 字典(dictionary)
  • 数组: 一组按次序排列的值，又称为序列(sequence) / 列表(list)
  • 纯量(scalars) : 单个的、不可再分的值
（1）对象
  • 对象键值对使用冒号结构表示key: value冒号后面要加一个空格。
  • 也可以使用key:{key1: value1, key2: value2， ...}
  • 还可以使用缩进表示层级关系;
（2）数组

以’-‘开头的行表示构成一个数组，如下效果类似于：[A,B,C]

YAML支持多维数组，可以使用行内表示

（3）常量

纯是最基本的，补可再分的值，包括:

  • 字符串
  • 布尔值
  • 整数
  • 浮点数
  • NULL
  • 时间
  • 日期
